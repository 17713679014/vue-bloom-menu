### get started
* 	git clone https://github.com/MingSeng-W/vue-bloom-menu.git
*  npm install
*  npm run dev
*  之后访问浏览器的8080端口，可以看到项目跑起来了(now please checkout the port 8080 in your browser, you will see the menu in page)
